Kinase search

Kinase search allows you to type your query kinase into the search bar.

This returns all kinase entries matching your query, taking into account gene name and aliases. If kinase name is unknown, you can search for part of the name or a single letter to return all entries featuring your query. From the table of results click on choosen kinase gene name to view more information.

Kinase Results

Selecting a kinase take you to the individual kinase information page.
Background information about the kinase is diplayed first. This shows information about:

  • Protein name
  • Uniprot number
  • Uniprot entry
  • Gene name
  • Kinase family
  • Gene aliases
  • Subcellular location
  • Inhibitor

To explore the uniprot page for your kinase simply click on the uniprot number.

Using tabs on the results table you can explore phosphosites the kinases acts on.

To find out more information about a substrate click on the substrate name.

Inhibitor search

Inhibitor search page includes a table with all inhibitors. Typing in the search bar will condense the table to results which feature the query term only.

Inhibitor Results

Clicking on chosen inhibitor name, you are taken to the individual inhibitor information page where you will find the following:

  • Chembl ID
  • Molecular weight
  • Smiles formular
  • Inchi formular
  • An image of the inhibitor
  • Inhibitor aliases
  • Target of the inhibitor(gene name and aliases)

Phosphosite search

Phosphosites can be searched by genomic location.

Alternatively you can search by substrate name.
